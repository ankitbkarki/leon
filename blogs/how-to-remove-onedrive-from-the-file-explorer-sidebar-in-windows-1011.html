How to Remove OneDrive from the File Explorer Sidebar in Windows 10/11

April 30, 2025

Let’s be honest—Microsoft OneDrive isn’t for everyone.

Some people love the idea of cloud syncing. Others... not so much. Maybe you already have a preferred cloud storage service like Google Drive or Dropbox. Maybe you just don’t want that extra icon cluttering your File Explorer sidebar. Or maybe, like many users, you never even set OneDrive up and yet it insists on staying there, staring at you from the left panel like an uninvited guest who’s made themselves comfortable on your couch.

Whatever your reason, removing OneDrive from the File Explorer sidebar in Windows 10 or 11 isn’t impossible—but it’s also not as straightforward as it probably should be.

This guide will walk you through everything. We'll cover a few different methods, depending on whether you want to just hide OneDrive, completely uninstall it, or remove only the shortcut from the File Explorer sidebar. There are nuances, so don’t worry if things get a little technical—we’ll keep it practical.

First, Let’s Clarify: What Exactly Are You Trying to Do?

This might seem like a silly question, but hear me out.

When people say they want to “remove OneDrive,” they often mean different things. Do you:

  • Just want the OneDrive icon gone from the File Explorer sidebar?

  • Want to stop OneDrive from syncing?

  • Want to uninstall OneDrive completely from your system?

  • Or maybe just disable it temporarily without fully removing it?

The reason it matters is because Windows treats OneDrive as a system component—and in some versions, particularly Windows 10 Home and Pro editions, it comes pre-installed and semi-glued into the OS. So how you approach this depends on how deep you want to go.

We’ll start with the most common goal: removing the OneDrive shortcut from the sidebar. Then we’ll look at how to disable or uninstall it completely, just in case you're going all in.

Method 1: Remove the OneDrive Shortcut from File Explorer Sidebar (via Registry Editor)

This is the cleanest way to get rid of that persistent OneDrive shortcut without affecting the app itself. It's just cosmetic—it doesn’t uninstall OneDrive or stop it from working. It just makes it stop showing up in File Explorer.

⚠️ Quick Disclaimer:

This method involves editing the Windows Registry. It's not difficult, but... you need to be careful. One typo and things can get messy. Back up your registry first—or at least create a system restore point.

Step-by-Step:

  1. Open the Registry Editor
    Press Windows + R to open the Run dialog. Type regedit and hit Enter.
    If you get a User Account Control prompt, click Yes.

  2. Navigate to the OneDrive Namespace Key
    Copy and paste the following path into the Registry Editor address bar:

    HKEY_CLASSES_ROOT\CLSID\

  3. Find the OneDrive CLSID Key
    Now here’s where it gets a bit... fiddly. You’re looking for a key that corresponds to OneDrive.
    For most systems, the CLSID is:

    {018D5C66-4533-4307-9B53-224DE2ED1FE6}

    On 64-bit Windows, you'll need to edit two locations:

    • HKEY_CLASSES_ROOT\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6}

    • HKEY_CLASSES_ROOT\Wow6432Node\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6}

  4. Modify the “System.IsPinnedToNameSpaceTree” Value
    Inside each key above, look for a DWORD value named System.IsPinnedToNameSpaceTree.

    • Double-click it.

    • Change the value data from 1 to 0.

    • Click OK.

  5. Restart File Explorer
    The changes won’t apply immediately. You can either restart your PC or restart File Explorer manually.

    To restart File Explorer:

    • Press Ctrl + Shift + Esc to open Task Manager.

    • Find Windows Explorer, right-click it, and choose Restart.

And just like that, the OneDrive shortcut should be gone from the sidebar.

Unless... it isn't. In which case, let’s move on.

Method 2: Unlink or Disable OneDrive (Without Uninstalling It)

Let’s say you’re not ready to uninstall it completely—you just want it dormant, not syncing, and ideally not bothering you.

You can unlink OneDrive from your Microsoft account. This stops it from syncing files without actually removing the app.

How to Unlink OneDrive:

  1. Right-click the OneDrive icon in your system tray (bottom-right corner).

  2. Click Settings (gear icon) > Accounts tab.

  3. Click Unlink this PC.

  4. Confirm.

Done. Your files won't sync anymore, and OneDrive will chill in the background without pestering you.

If you also want to stop it from starting up every time you boot your PC:

  • Still in the OneDrive settings window, go to the Settings tab.

  • Uncheck "Start OneDrive automatically when I sign in to Windows."

Method 3: Uninstall OneDrive Completely (If You're Sure)

Alright. So you’ve decided OneDrive is really not for you. Fair enough.

If you’re using Windows 10 or 11, you can uninstall OneDrive—though Microsoft doesn’t make it super obvious how to do it. Fortunately, it's not hard.

Option A: Uninstall via Settings

  1. Press Windows + I to open Settings.

  2. Go to Apps > Installed Apps (or Apps & Features in older versions).

  3. Find Microsoft OneDrive in the list.

  4. Click the three dots (or just click it), then choose Uninstall.

That should do it.

Option B: Use Command Prompt (for Stubborn Cases)

If for some reason the normal uninstall doesn’t work (or the app doesn’t show up), you can do it manually via Command Prompt.

For Windows 64-bit:

taskkill /f /im OneDrive.exe
%SystemRoot%\SysWOW64\OneDriveSetup.exe /uninstall

For Windows 32-bit:

taskkill /f /im OneDrive.exe
%SystemRoot%\System32\OneDriveSetup.exe /uninstall

Paste those into an elevated Command Prompt (run it as Administrator), and OneDrive will be gone.

Now, does this also remove the File Explorer icon? Sometimes, yes. But sometimes... no. Windows is weird like that.

If the icon lingers even after uninstalling, go back and use the Registry method from earlier. The combo usually does the trick.

Bonus: Prevent OneDrive from Reinstalling via Windows Updates

Here's a fun twist. Even if you uninstall OneDrive, a big Windows update might quietly reinstall it without asking. Not always—but it’s happened.

There’s no bulletproof way to block this unless you're using Group Policy (available only in Pro or Enterprise editions). Still, here’s a preventative step:

Use Group Policy Editor (Windows Pro)

  1. Press Windows + R, type gpedit.msc, press Enter.

  2. Go to:
    Computer Configuration > Administrative Templates > Windows Components > OneDrive

  3. Double-click Prevent the usage of OneDrive for file storage.

  4. Set it to Enabled, click Apply, then OK.

This not only prevents usage—it usually blocks the app from being reinstalled automatically.

Unfortunately, this method won’t work on Windows Home editions. There are workarounds (involving adding Group Policy manually), but they’re finicky, and honestly, if you’re using Home, the registry tweaks will usually suffice.

So... Should You Remove OneDrive?

Maybe. Maybe not.

If you’re not using it, it’s reasonable to want it out of the way. It's not particularly lightweight, and its constant presence can feel a bit pushy—especially if you already use other services. That said, OneDrive can be useful. It integrates tightly with Windows, backs up your files, and plays nice with Office apps.

Still, if it’s just the File Explorer shortcut bugging you? The registry method is your best bet. Low risk, reversible, and it doesn’t mess with anything else.

If you want the whole thing gone? Go ahead and uninstall it—but keep an eye out after big Windows updates. Microsoft has a habit of bringing back the things you tried to ditch.

A Final Thought (or Two)

It’s strange that Microsoft doesn’t offer a simple “Hide OneDrive from File Explorer” toggle in settings. You’d think that’d be standard. Instead, we’re stuck editing registry keys and running command line scripts like it’s 1998.

Still, there’s something oddly satisfying about reclaiming control over your own interface—even if it takes a few hoops.

So whether you’re doing a little digital spring cleaning or just have an aversion to unnecessary icons, now you’ve got a few solid tools to deal with it. Take what you need, skip what you don’t. And hey, if you ever change your mind, OneDrive’s just a download away.
